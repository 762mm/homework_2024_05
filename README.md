# Домашняя работа

### Задание
Данные справочника городов и реестра субъектов приведены во вложении, требуется вывести данные (Наименование, Форма, Код города, Название города) обо всех версиях сведений о субъектах из реестра в хронологическом порядке для дальнейшей аналитики.
Данные хранятся в отдельных csv-файлах, необходимо реализовать полноценный скрипт чтения-записи данных средствами python

### Решение
Исходные данные находятся в директории "source" в файлах "register.csv" и "referenc.csv", результат записывается в файл "result.csv" в директории "result".
Перед очередным запуском файл с предыдущими результатами желательно удалить - проверка на наличие старого файла отсутствует.

### Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone git@github.com:762mm/homework_2024_05.git
```

```
cd homework_2024_05
```

Cоздать и активировать виртуальное окружение:

- Если у вас Linux/macOS
    ```
    python3 -m venv env
    ```

    ```
    source env/bin/activate
    ```

- Если у вас Windows
    ```
    python -m venv venv
    ```

    ```
    source venv/scripts/activate
    ```

Запустить проект из строки терминала командой
    ```
    python homework.py
    ```
